## 🚀 1683. Invalid Tweets

### 💡 Idea

The goal is to find all tweets whose content is **longer than 15 characters**.

We use MySQL's `CHAR_LENGTH()` function to determine the number of characters in each tweet and filter those exceeding the limit.

---

## 💻 Solution

```sql
SELECT tweet_id
FROM Tweets
WHERE CHAR_LENGTH(content) > 15;
```

---

### 🛠️ How It Works

1. Read each tweet from the `Tweets` table.
2. Use `CHAR_LENGTH(content)` to calculate the number of characters in the tweet.
3. Keep only the tweets whose length is greater than **15**.
4. Return the corresponding `tweet_id`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n × m)`  
*(`n` = number of tweets, `m` = average tweet length)*

**Space Complexity:** `O(1)`
