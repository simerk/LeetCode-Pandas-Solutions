## 🚀 1789. Primary Department for Each Employee

### 💡 Idea

The goal is to determine the **primary department** for each employee.

- If an employee belongs to **only one department**, that department is considered the primary department.
- If an employee belongs to **multiple departments**, the row where `primary_flag = 'Y'` represents the primary department.

We can identify both cases by grouping employees and using a `HAVING` clause.

---

## 💻 Solution

```sql
SELECT employee_id, department_id
FROM Employee
WHERE primary_flag = 'Y'
UNION
SELECT employee_id, department_id
FROM Employee
GROUP BY employee_id
HAVING COUNT(*) = 1;
```

---

### 🛠️ How It Works

1. Select employees whose `primary_flag = 'Y'`.
2. Select employees who belong to only one department using:
   - `GROUP BY employee_id`
   - `HAVING COUNT(*) = 1`
3. Combine both result sets using `UNION`.
4. Since `UNION` removes duplicates, employees appearing in both queries are returned only once.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(two scans of the table)*

**Space Complexity:** `O(n)` *(stores the union result)*
