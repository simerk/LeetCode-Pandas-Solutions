## 🚀 1527. Patients With a Condition

### 💡 Idea

The goal is to find all patients whose `conditions` column contains a condition code that **starts with `DIAB1`**.

A patient is considered diabetic if the condition appears:

- At the **beginning** of the string, or
- After a **space**, indicating the start of another condition code.

We can use the `LIKE` operator to check for both cases.

---

## 💻 Solution

```sql
SELECT * FROM Patients
WHERE conditions LIKE 'DIAB1%' OR conditions LIKE '% DIAB1%';
```

---

### 🛠️ How It Works

1. `conditions LIKE 'DIAB1%'` finds patients whose first condition starts with `DIAB1`.
2. `conditions LIKE '% DIAB1%'` finds patients where a condition starting with `DIAB1` appears after a space.
3. Combine both conditions using `OR` to return all matching patients.

---

### ⏱️ Complexity

**Time Complexity:** `O(n × m)`  
*(`n` = number of patients, `m` = average length of the `conditions` string)*

**Space Complexity:** `O(1)`
