## 🚀 1587. Bank Account Summary II

### 💡 Idea

The goal is to find the users whose **total account balance is greater than 10000**.

We join the `Users` and `Transactions` tables using the account number, calculate the total balance for each user by summing their transaction amounts, and then filter users whose balance exceeds **10000**.

---

## 💻 Solution

```sql
SELECT u.name, SUM(t.amount) AS balance
FROM Users u
INNER JOIN Transactions t
ON u.account = t.account
GROUP BY u.account, u.name
HAVING SUM(t.amount) > 10000;
```

---

### 🛠️ How It Works

1. Join the `Users` and `Transactions` tables using the `account` column.
2. Group the records by user.
3. Calculate each user's account balance using `SUM(amount)`.
4. Use the `HAVING` clause to keep only users whose balance is greater than **10000**.
5. Return the user's name and balance.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single join followed by grouping)*

**Space Complexity:** `O(n)` *(stores grouped users)*
