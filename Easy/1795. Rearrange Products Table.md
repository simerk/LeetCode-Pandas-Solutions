## 🚀 1795. Rearrange Products Table

### 💡 Idea

The `Products` table stores prices for different stores in separate columns:

- `store1`
- `store2`
- `store3`

The goal is to transform this wide format into a long format with three columns:

- `product_id`
- `store`
- `price`

We can achieve this by selecting each store separately and combining the results using `UNION ALL`. Rows with `NULL` prices are excluded.

---

## 💻 Solution

```sql
SELECT product_id,
'store1' AS store,
store1 AS price
FROM Products
WHERE store1 IS NOT NULL
UNION ALL
SELECT product_id,
'store2' AS store,
store2 AS price
FROM Products
WHERE store2 IS NOT NULL
UNION ALL
SELECT product_id,
'store3' AS store,
store3 AS price
FROM Products
WHERE store3 IS NOT NULL;
```

---

### 🛠️ How It Works

1. Select the `product_id`, store name, and price for **store1**.
2. Exclude products whose `store1` price is `NULL`.
3. Repeat the same process for **store2** and **store3**.
4. Combine all three result sets using `UNION ALL`.
5. The final result contains one row for every product-store combination with a valid price.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(three scans of the table, one for each store column)*

**Space Complexity:** `O(n)` *(stores the transformed rows)*
