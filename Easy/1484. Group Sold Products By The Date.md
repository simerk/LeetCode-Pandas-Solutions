## 🚀 1484. Group Sold Products By The Date

### 💡 Idea

The goal is to generate a report for each selling date containing:

- The number of **distinct** products sold.
- A comma-separated list of those products in **lexicographical order**.

We group the records by `sell_date`, count the unique products, and use MySQL's `GROUP_CONCAT()` function to combine the product names into a single string.

---

## 💻 Solution

```sql
SELECT sell_date, COUNT(DISTINCT product) AS num_sold,
GROUP_CONCAT(
  DISTINCT product
  ORDER BY product
  SEPARATOR ','
) AS products
FROM Activities
GROUP BY sell_date
ORDER BY sell_date;
```

---

### 🛠️ How It Works

1. Group the records by `sell_date`.
2. Count the number of **distinct** products sold on each date using `COUNT(DISTINCT product)`.
3. Use `GROUP_CONCAT()` to combine all distinct product names into a single comma-separated string.
4. Sort the product names alphabetically using `ORDER BY product` inside `GROUP_CONCAT()`.
5. Return the results ordered by `sell_date`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(grouping plus sorting product names within each group)*

**Space Complexity:** `O(n)` *(stores grouped products and concatenated strings)*
