## 🚀 2356. Number of Unique Subjects Taught by Each Teacher

### 💡 Idea

The goal is to find the number of **unique subjects** taught by each teacher.

Since a teacher may teach the same subject in multiple departments, we use `COUNT(DISTINCT subject_id)` to count each subject only once.

Finally, group the records by `teacher_id`.

---

## 💻 Solution

```sql
SELECT teacher_id, COUNT(DISTINCT subject_id) AS cnt
FROM Teacher
GROUP BY teacher_id;
```

---

### 🛠️ How It Works

1. Group the records by `teacher_id`.
2. Use `COUNT(DISTINCT subject_id)` to count the unique subjects taught by each teacher.
3. Return the teacher ID along with the count of unique subjects.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan followed by grouping)*

**Space Complexity:** `O(n)` *(stores grouped teachers)*
