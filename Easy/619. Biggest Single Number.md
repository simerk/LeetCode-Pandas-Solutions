## 🚀 619. Biggest Single Number

### 💡 Idea

The goal is to find the **largest number that appears exactly once** in the `MyNumbers` table.

First, group the numbers and keep only those that occur **once** using `HAVING COUNT(*) = 1`. Then, return the largest of these numbers using `MAX()`. If there is no single number, `MAX()` automatically returns `NULL`, as required.

---

### 💻 Solution

```sql
SELECT MAX(num) AS num
FROM (
    SELECT num FROM MyNumbers
    GROUP BY num
    HAVING COUNT(*) = 1
) AS t;
```

---

### 🛠️ How It Works

1. Group the records by `num`.
2. Count the occurrences of each number.
3. Keep only the numbers that appear exactly once using `HAVING COUNT(*) = 1`.
4. Pass the remaining numbers to the outer query.
5. Use `MAX()` to return the largest single number.
6. If no number appears exactly once, `MAX()` returns `NULL`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan for grouping and aggregation)*

**Space Complexity:** `O(n)` *(stores grouped numbers)*
