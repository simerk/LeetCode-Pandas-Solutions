## 🚀 2877. Create a DataFrame from List

### 💡 Idea

The goal is to create a **Pandas DataFrame** from a given Python list named `student_data`.

Each element in the list represents a student's record containing:

- `student_id`
- `age`

We simply pass the list to `pd.DataFrame()` and specify the column names.

---

## 💻 Solution

```python
import pandas as pd

def createDataframe(student_data: list[list[int]]) -> pd.DataFrame:
    return pd.DataFrame(
        student_data,
        columns=["student_id", "age"]
    )
```

---

### 🛠️ How It Works

1. The input `student_data` is a list of lists.
2. Pass the list to `pd.DataFrame()`.
3. Specify the column names:
   - `student_id`
   - `age`
4. Return the resulting DataFrame.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)`

*(`n` = number of rows in the input list)*

**Space Complexity:** `O(n)`
