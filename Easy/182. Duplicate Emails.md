## 🚀 182. Duplicate Emails

### 💡 Idea

To find duplicate emails, group all records by the `email` column and count how many times each email appears.

If an email appears more than once, it is considered a duplicate.

---

### 💻 Solution

```sql
SELECT email
FROM Person
GROUP BY email
HAVING COUNT(email) > 1;
```

---

### 🛠️ How It Works

1. Group all records by the `email` column.
2. Count the number of occurrences of each email using `COUNT()`.
3. Use the `HAVING` clause to filter only those groups where the count is greater than `1`.
4. Return the duplicate email addresses.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan with grouping, depending on indexing)*

**Space Complexity:** `O(n)` *(stores grouped email values during aggregation)*
