## 🚀 610. Triangle Judgement

### 💡 Idea

For each row in the `Triangle` table, we need to determine whether the three side lengths (`x`, `y`, and `z`) can form a triangle.

A valid triangle must satisfy the **Triangle Inequality Theorem**:

* The sum of any two sides must be greater than the third side.
* Equivalently, it is sufficient to check:

  * `x + y > z`
  * `x + z > y`
  * `y + z > x`

If all three conditions are true, the answer is **'Yes'**; otherwise, it is **'No'**.

---

### 💻 Solution

```sql
SELECT *, CASE
    WHEN (x + y) > z AND (y + z) > x AND (z + x) > y
    THEN 'Yes'
    ELSE 'No'
    END AS triangle
from Triangle;
```

---

### 🛠️ How It Works

1. Read each row from the `Triangle` table.
2. Use a `CASE` expression to check the three triangle inequality conditions.
3. If all three conditions are satisfied, return `'Yes'`.
4. Otherwise, return `'No'`.
5. Output the original side lengths along with the result.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(processes each row exactly once)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
