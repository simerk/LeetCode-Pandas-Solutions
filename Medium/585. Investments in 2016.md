## 🚀 585. Investments in 2016

### 💡 Idea

The goal is to find the **total investment value in 2016 (`tiv_2016`)** for policyholders who satisfy **both** of the following conditions:

- Their `tiv_2015` value is **shared with at least one other policyholder**.
- Their location (`lat`, `lon`) is **unique**.

We can identify these conditions using two subqueries with `GROUP BY` and `HAVING`, then sum the qualifying `tiv_2016` values.

---

### 💻 Solution

```sql
SELECT ROUND(SUM(tiv_2016), 2) AS tiv_2016
FROM Insurance
WHERE tiv_2015 IN (
    SELECT tiv_2015
    FROM Insurance
    GROUP BY tiv_2015
    HAVING COUNT(*) > 1
)
AND (lat, lon) IN (
    SELECT lat, lon
    FROM Insurance
    GROUP BY lat, lon
    HAVING COUNT(*) = 1
);
```

---

### 🛠️ How It Works

1. Find all `tiv_2015` values that appear more than once.
2. Find all unique `(lat, lon)` location pairs.
3. Filter the `Insurance` table to keep only records that satisfy both conditions.
4. Sum the corresponding `tiv_2016` values.
5. Round the final result to **2 decimal places**.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(multiple grouping operations with linear scans)*

**Space Complexity:** `O(n)` *(stores grouped values for `tiv_2015` and `(lat, lon)` combinations)*
